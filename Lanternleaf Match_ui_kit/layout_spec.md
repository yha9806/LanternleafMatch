# Whisperwood Match — Mobile Portrait UI Spec (v0.1)

## Screen (1080×1920 reference)
- Top Safe/Status: 6–8%
- Story Panel: 22–28%
- Board: 46–54% (6×6)
- Bottom Controls: 10–14%

## Core Components
1) StoryCard (top)
- Line1: 1 sentence (<= 18 CN chars)
- Line2: optional (tap to expand)
- MiniGoalHint: icon + count

2) HUD Bar (above board)
- Level badge / Goal widget / Moves widget / Pause button

3) BoardGrid (6×6)
- Tile size: 128px (reference), with 8–12px padding between tiles

4) Bottom Controls
- Skill button (Cat/Dog)
- Energy mini indicator (tap opens EnergyGate)

## Modal States
- PauseModal / WinModal / LoseModal / EnergyGateModal / ConsentModal
